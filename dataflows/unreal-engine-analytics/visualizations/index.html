Unreal Engine: Infinyon Analytics Plugin

This pase is visualizing data collected by the sample UE5 Tudi Analytics game via the Infinyon Analytics Plugin. The fluvio cluster is using sdf dataflows to process the data. The dataflow uses a generic sdf analytics package to parse the events, enhance them with added notations, and then output them via the Infinyon Web socket gateway.

Individual game session traces are output here, colors are rotated per game session
